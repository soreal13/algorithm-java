
## 알고리즘 풀이 관련한 기본 문법 사항들

### 입력 받기

####1.Scanner
    Scanner in = new Scanner(System.in);  
``` in.nextInt()```를 많이 씀. ```in.next()```의 경우 라인 한 줄이 받아지는 것에 유의.

####2. BufferReader
    BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
````Integer.parseInt(br.readLine())````
